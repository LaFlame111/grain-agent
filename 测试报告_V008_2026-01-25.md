## 问题与建议
1. **WMS 数据稀疏导致图表生成失败**
   - `test_requests.py`、`test_time_range_query.py`、`test_v007_data.py` 在 2026-01-01 到 2026-01-05 仅 1 条记录，三温图/两湿图返回 `no_data`
   - 建议：选择数据点更密集的时间区间，或放宽图表最小点数阈值
2. **WMS API 巡检工具 404**
   - `test_agent.py` 的巡检用例会触发对 `Q1-S1` 等 Mock ID 的实时查询，返回 `404`
   - 建议：巡检工具应支持时间范围参数，或在测试环境中使用真实存在的传感器 ID
3. **交互式测试可自动化**
   - `test_requests_qa.py` 需要输入，当前已通过预置输入跑通一次查询
   - 建议：为该脚本提供非交互模式参数，便于回归自动化
